# Gestión de Alumnos

## Descripción
Sistema de gestión de alumnos desarrollado en C# con WinForms. Permite:
- Gestión CRUD de alumnos.
- Cálculo de edad de los alumnos (considerando día y mes).
- Cálculo de cantidad de materias no aprobadas.

## Tecnologías Utilizadas
- Lenguaje: C#
- IDE: Visual Studio
- Framework: WinForms (.NET Framework)
- Base de datos: Excel CSV (en desarrollo)

## Instalación y uso
1. Cloná este repositorio:
	```bash
    git clone https://github.com/WladekDev/GestionAlumnos.git
2. Abrí el proyecto con Visual Studio.
3. Ejecutá el archivo "Sistema Alumnos.sln".
4. Ingresá datos en los campos.
5. Explorá las funcionalidades del programa.
